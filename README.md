# E-Commerce Data Analysis Dashboard ✨

## Setup Environment - Anaconda (Conda)
1. Buka Anaconda Prompt atau Terminal.
2. Buat virtual environment baru dengan nama `proyek-analisis-data` :
   ```bash
   conda create --name proyek-analisis-data python=3.9
   conda activate proyek-analisis-data
   pip install -r requirements.txt
   ```
## Run Streamlit App
```bash
streamlit run dashboard/dashboard.py
```
